Создал локацию симметричную локацию для мультиплеера с помощью примитивов Unity: кубов, цилиндров, капсул и сфер, используя фотографии Мурманска: <img width="1330" height="747" alt="Снимок экрана 2026-04-20 в 10 06 40" src="https://github.com/user-attachments/assets/20e4838c-dbc0-457c-b7d6-764699ad8339" />
Создал префабы (см. папку Prefabs): окна домов, балкон, застекленный балкон, двери, гараж, вентиляционный короб, два вида деревьев, забор, домофон, пожарная лестница, фонарь освещения подьезда, 
Использовал два pakage из Unity Asset Store Tokyo_Street и FlamingSands_Cars для декорирования локации реалистичными предметами.
Настроил освещение в соответствии со временем суток: днём, вечером и ночью.
Создал несколько источников света: от фонарей над подъездом, окон домов и фар автомобилей.
